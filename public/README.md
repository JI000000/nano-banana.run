# 静态资源说明

本目录包含网站的静态资源文件。

## 重要文件

- **favicon.ico**: 网站图标 (需要替换为实际的Nano-Banana图标)
- **robots.txt**: 搜索引擎爬虫规则
- **images/**: 图片资源目录

## 上线前检查清单

1. 替换favicon.ico为高质量的网站图标
2. 确保所有图片资源已经准备就绪
3. 验证robots.txt和sitemap.xml配置正确

## 图标要求

favicon需要满足以下要求:
- 尺寸: 32x32像素 (基础尺寸)
- 格式: .ico (包含多个尺寸变体)
- 也可以提供以下格式以支持更多平台:
  - apple-touch-icon.png (180x180)
  - favicon-32x32.png
  - favicon-16x16.png
  - site.webmanifest (Web App Manifest)
